# ميزات البائع - لوحة التحكم

## إضافة المنتجات

### الفئات المدعومة

#### 1. **Shop (المنتجات اليدوية)**
- **الوصف**: منتجات يدوية مصنوعة يدوياً
- **الحقول المطلوبة**:
  - عنوان المنتج
  - الوصف
  - السعر
  - صورة المنتج (اختياري)
  - الكمية المتوفرة

#### 2. **Learn (الدورات التعليمية)**
- **الوصف**: دورات تعليمية وتutorials
- **الحقول المطلوبة**:
  - عنوان الدورة
  - الوصف
  - السعر
  - صورة الدورة (اختياري)
  - مدة الدورة
  - عدد الطلاب الحاليين

#### 3. **Materials (المواد الخام)**
- **الوصف**: مواد وأدوات للحرف اليدوية
- **الحقول المطلوبة**:
  - عنوان المادة
  - الوصف
  - السعر
  - صورة المادة (اختياري)
  - العلامة التجارية
  - الكمية المتوفرة

## كيفية الاستخدام

### 1. تسجيل الدخول كبائع
```
البريد: seller1@arty.com
كلمة المرور: seller123
```

### 2. الوصول للوحة التحكم
- سجل دخول بحساب البائع
- ستتم توجيهك تلقائياً لـ `seller-dashboard.html`

### 3. إضافة منتج جديد
1. **اذهب لـ "My Products" tab**
2. **اضغط "+ Add New Product"**
3. **املأ النموذج**:
   - اختر الفئة (Shop/Learn/Materials)
   - املأ المعلومات الأساسية
   - املأ الحقول الخاصة بالفئة المختارة
4. **اضغط "Add Product"**

### 4. إدارة المنتجات
- **عرض المنتجات**: جميع منتجاتك مع تفاصيلها
- **تعديل المنتجات**: زر Edit لكل منتج
- **حذف المنتجات**: زر Delete لكل منتج
- **تصنيف المنتجات**: badges ملونة لكل فئة

## الميزات الإضافية

### إحصائيات المبيعات
- **عدد المنتجات**: إجمالي منتجاتك
- **إجمالي المبيعات**: عدد الطلبات
- **الأرباح**: إجمالي الأرباح
- **التقييم**: متوسط تقييم منتجاتك

### إدارة الطلبات
- **عرض الطلبات**: جميع الطلبات التي تحتوي على منتجاتك
- **حالة الطلبات**: معلق، مكتمل، ملغي
- **تفاصيل العملاء**: معلومات العملاء الذين طلبوا منتجاتك

### الملف الشخصي
- **معلومات المتجر**: اسم المتجر والوصف
- **تاريخ الانضمام**: متى انضممت للمنصة
- **إحصائيات شاملة**: مبيعات وأرباح

## نصائح للبائعين

### 1. **صور المنتجات**
- استخدم صور عالية الجودة
- تأكد من أن الصور تعرض المنتج بوضوح
- استخدم إضاءة جيدة

### 2. **الوصف**
- اكتب وصفاً مفصلاً ومفيداً
- اذكر المواد المستخدمة
- اذكر الأبعاد والأحجام

### 3. **الأسعار**
- ادرس أسعار المنافسين
- احسب تكلفة الإنتاج
- اترك هامش ربح معقول

### 4. **المخزون**
- حافظ على تحديث الكميات المتوفرة
- خطط للمخزون مسبقاً
- أزل المنتجات غير المتوفرة

## الدعم الفني

إذا واجهت أي مشاكل:
1. تأكد من تسجيل الدخول بحساب البائع
2. تحقق من اكتمال جميع الحقول المطلوبة
3. تأكد من صحة رابط الصورة
4. امسح cache المتصفح إذا لزم الأمر
